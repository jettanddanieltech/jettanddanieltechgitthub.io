AGI: Artificial General Intelligence. AI that can perform any intellectual task a human can.[LEVEL_5]
BACKPROPAGATION: The primary algorithm for training neural networks by calculating gradients.[NEURAL]
CHIPLET: Small integrated circuits that make up a larger processor, vital for AI hardware scalability.
DEEP_LEARNING: A subset of ML based on artificial neural networks with multiple layers.
ENCODER: A component that converts input data into a hidden representation (latent space).
FINE_TUNING: Taking a pre-trained model and training it further on a specific dataset.
GAN (GENERATIVE ADVERSARIAL NETWORK): Two neural networks competing to create realistic data (images/video).
HALLUCINATION: When an AI generates confident but false or illogical information.
INFERENCE: The process of a trained AI model making predictions on new data.
JUPYTER_NOTEBOOK: An open-source web app used to create and share documents containing live AI code.
K-NEAREST_NEIGHBORS: A simple ML algorithm used for classification and regression.
LATENT_SPACE: A compressed representation of data where similar items are mapped closer together.
MLOps: A set of practices that aims to deploy and maintain machine learning models in production reliably.
NLP: Natural Language Processing. Helping computers understand and interpret human language.
OVERFITTING: When a model learns training data too well, failing to generalize to new data.
PARAMETER: A variable in an AI model that is learned from the data during training.
QUANTIZATION: Reducing the precision of AI model weights to make them run faster on hardware.
REINFORCEMENT_LEARNING: Training AI through a system of rewards and punishments.
TRANSFORMER: The architecture behind GPT models that uses "attention" to process sequences.
ZERO-SHOT_LEARNING: An AI's ability to complete a task it wasn't specifically trained for.
