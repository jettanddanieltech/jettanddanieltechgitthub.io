TECH_HARDWARE_DICTIONARY

ARM_ARCHITECTUREA type of CPU architecture known for energy efficiency, used in smartphones and M-series Macs.
BITRATEThe amount of data processed over time in a video or audio file; higher means better quality.
CLOCK_SPEEDThe speed at which a microprocessor executes instructions, measured in Gigahertz (GHz).
DDR5The latest generation of Double Data Rate RAM, providing higher speeds and efficiency.
ETHERNETThe standard technology for connecting devices in a wired local area network (LAN).
FIRMWAREPermanent software programmed into a read-only memory of a hardware device.
GPUGraphics Processing Unit. A specialized circuit designed to accelerate image creation and AI tasks.
HOT_SWAPPABLEThe ability to replace components (like hard drives) without shutting down the system.
IP_ADDRESSA unique string of numbers separated by periods that identifies each computer using the Internet Protocol.
JSONJavaScript Object Notation. A lightweight data-interchange format easy for humans to read/write.
KERNELThe core of an operating system that manages operations of computer and hardware.
LAN_PARTYA gathering of people with computers between which they establish a local area network for gaming.
MOTHERBOARDThe main printed circuit board (PCB) in general-purpose computers.
NVMeNon-Volatile Memory Express. A protocol for accessing high-speed storage media like SSDs.
OVERCLOCKINGIncreasing the clock rate of a component to exceed the manufacturer's specified speed.
PENTESTINGPenetration Testing. The practice of testing a computer system to find security vulnerabilities.
RAIDRedundant Array of Independent Disks. Combining multiple disk drives for data redundancy or speed.
SATASerial AT Attachment. An interface used to connect SSDs and HDDs to the motherboard.
UEFIUnified Extensible Firmware Interface. A modern replacement for the legacy BIOS.
VRAMVideo RAM. Dedicated memory used by the GPU to store image data and textures.
