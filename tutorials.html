01. CRACKED SCREEN RECOVERY

! WARNING: WEAR EYE PROTECTION WHEN HANDLING SHATTERED GLASS !
PROBLEM: Display is cracked or touch is unresponsive.
FIX: Tape the glass, heat the edges to soften adhesive, and use a thin pry tool to lift the panel. Disconnect ribbons before full removal.

02. LAGGING PC CLEANUP

PROBLEM: High CPU usage and slow boot times.
FIX: Clear %temp%, disable startup apps in Task Manager, and run Disk Cleanup as Admin.

03. GITHUB REPO RESET

PROBLEM: Broken 404 links or long folder URLs.
FIX: Rename repo to jettanddanieltech in Settings. Delete CNAME if no custom domain is active.

04. RECOVERING "BRICKED" PHONES

PROBLEM: Phone stuck on boot logo.
FIX: Perform a "Forced Restart" (Vol Up, Vol Down, then hold Power). If that fails, enter DFU mode via PC.

05. CSS NEON GLOW EFFECT

PROBLEM: Design looks too plain.
FIX: Use text-shadow: 0 0 10px #33BBFF; to create the J&D signature glow.

06. WATER DAMAGE FIRST AID

! DO NOT PUT IT IN RICE - RICE IS A MYTH !
PROBLEM: Device dropped in liquid.
FIX: Power off immediately. Open the device and use 90% Isopropyl Alcohol to clean corrosion. Use a fan, not a hairdryer.

07. CHARGING PORT RESCUE

PROBLEM: Cable won't stay in or phone charges "slowly."
FIX: Use a wooden toothpick or plastic pick to gently scrape out pocket lint. Never use metal pins as they short the pins!

08. THERMAL PASTE RE-APP

PROBLEM: Laptop or Desktop hitting 90°C+ (Overheating).
FIX: Remove the heatsink, clean old "crusty" paste with alcohol, and apply a pea-sized drop of new thermal compound.
